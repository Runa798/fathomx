# ADR-0003: 首攻能力 = 竞品深度研究

Status: Accepted (2026-05-29)

## Context
FathomX 目标含 ≥4 类能力（竞品/产品能力/创新方向/需求调研）。需要选一个先端到端跑通+验证，作为 v2.0 的第一份真实用户价值，避免四线并进都做不深。

## Decision
**先做竞品深度研究**，作为 v2.0。其余能力（产品能力研究、创新方向研究、产品需求深度调研）在 Phase 3 后续逐版加（v2.1/v2.2/v2.3）。

## Consequences
- 上轮已用教育类产品竞品调研 dry-run 过 Lapis，最熟、需求最普遍。
- Phase 2 的评测 rubric + 黄金样例优先围绕竞品研究构建。
