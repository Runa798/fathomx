# 计划目录 (rolling-wave)

每个 Phase / 阶段**开始**时在此新建细化计划；**完成**时在同文件追加 review 小节，并回写 `../../ROADMAP.md`。

命名：`phaseN-<slug>.md`（如 `phase1-method-decisions.md`）；阶段细分用 `phaseN-stageM-<slug>.md`。

计划文件建议结构：目标 / 输入 / 任务分解 / 产出物 / 退出标准 / 风险 / （完成后）Review。
